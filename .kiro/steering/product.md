# Product Overview

CLI ベースの「キャリア適合度チェッカー」。貼り付けテキストやファイル入力からプロフィールを組み立て、将来の求人評価のための下地を作る。現状はプロフィール生成とバリデーションの骨格を MVP として整備中。

## Core Capabilities

- Typer を使ったプロフィール管理 CLI（profile create/update/show の外形を提供）
- 入力テキストをチャンク化し、セッション辞書でワークフローに受け渡す仕組み
- プロフィールスキーマのドラフト/確定バリデーション（欠損検出を明示）
- LangGraph を用いたステートフルなフロー構築（未実装ノードは警告で可視化）

## Target Use Cases

- 自己紹介や経歴をテキストで入力し、プロフィール JSON を生成・点検する
- 将来の求人評価ワークフローの土台として、プロフィール状態を安全に保持する
- CLI で非対話/対話を切り替えながらプロフィールを整備する

## Value Proposition

- 「貼り付けテキスト→構造化プロフィール」を最短経路で作るための軽量 CLI
- 欠損項目を検知して明示し、後続の AI 生成/レビューに渡しやすいデータ形にする
- LangGraph ベースの拡張余地を残しつつ、MVP で必要な最小の型とフローを提供
