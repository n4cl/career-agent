career-agent
===

## 概要

`career-agent` は、あなたのプロフィールと求人情報を基に、求人との適合性を評価するツールです。
これにより、客観的な視点からキャリア選択の意思決定を支援します。

## 主な機能（予定）

- プロフィール管理
  - `profile update --file <path>`: 職務経歴書ファイルからプロフィールを更新します。
  - `profile show`: 現在のプロフィール情報を表示します。
- 求人情報の解析
  - `job parse --file <path>`: 求人情報ファイルから構造化データを抽出します。
- 求人情報の評価
  - `evaluate --job <path>`: 求人情報を評価し、レポートを作成します。
- 評価レポートの管理
  - `report list`: 作成済みの評価レポートの一覧を表示します。
  - `report show <report_name>`: 指定されたレポートの内容を表示します。
